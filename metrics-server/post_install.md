## Metrics Server - provide metrics for Kubernetes HPAv2 (Pod auto-scaling)

### Test it out

Check that data is being received from nodes:

```
kubectl get --raw "/apis/metrics.k8s.io/v1beta1/nodes"
```

Check node usage:

```sh
kubectl top node
```

Check node usage:

```sh
kubectl top pod --all-namespaces
```
